# Booktopia
<h3>Booktopia is a Python based online library, where you can read, write and know all about your favorite books!</h3>
<h4>This online library has five main features:</h4>
<ol><li>Account register/login: The user can create their own account and login later.</li>
<li>Reading space: This is where the pdfs' of some famous books, that can be downloaded and read, are available.</li>
<li>Writing space: The user can write down their thoughts, or a summary of the book that they read in this writing space. Not only this, but they can also save this as a text file in their systems.</li>
<li>Fandom: This exciting feature provides the user with some interesting facts about their favourite book series.</li>
<li>Feedback: The user can also provide a feedback!</li></ol>
I created this class 12 python project using tkinter and some basic knowledge on file handling.
